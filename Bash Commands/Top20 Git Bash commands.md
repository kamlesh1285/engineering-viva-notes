1. pwd       [print working Directory]
 * fn:- Display the absolute path of your current directory.
2. ls -la    
 * fn:- Lists all files and folders, including hidden files, with detailed information.
3. cd       [Change Directory]
 * fn:- Moves into another folder 
4. mkdir    [Make Directory] 
 * fn:- creates a new folder 
5. touch 
 * fn:- creates an empty file 
6. code . 
 * fn:- opens the current folder in visual studio code.
7. git init 
 * fn:- initialize a git repository in the current folder 
8. git status 
 * fn:- shows the current state of your repository 
9. git add .  
 * fn:- stages all changes in the current directory for the next commit.
10. git commit -m "..." 
 * fn:- saves a snapshot of the staged changes in Git history.
11. git branch -M main 
 * fn:- rename the current branch to main.
12. git remote add origin main 
 * fn:- connects your local repo to a remote repository on Github.
13. git push -u origin main 
 * fn:- uploads yours local commits to github 
14. git pull 
 * downloads and merge the latest changes from the remote repository into your local branch.
