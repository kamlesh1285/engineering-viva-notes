Experiment 5: Array and Object Operations using JavaScript

What is an Array?
An ordered collection of values indexed by integers, used to store lists.

What is an Object?
A collection of key-value pairs (properties) that model entities and their behaviors.

What is the difference between Arrays and Objects?
Arrays are ordered lists accessed by numeric indices; objects are unordered maps accessed by keys (strings/symbols).

What is the purpose of filter()?
Returns a new array containing elements that pass a provided test function.

What is the purpose of forEach()?
Executes a provided function once for each array element, typically for side effects.

What is the purpose of map()?
Creates a new array by transforming each element with a provided function.

What is the purpose of reduce()?
Reduces an array to a single value by cumulatively applying a reducer function.

How are employee records stored?
Often as an array of objects, each object representing an employee with properties like name, id, salary.

What is object property access?
Getting or setting values using dot notation (obj.prop) or bracket notation (obj['prop']).

What is dynamic data display?
Rendering data-driven content in the UI that updates when data changes.

How is average salary calculated?
Sum all employee salaries then divide by number of employees; can use reduce to compute the sum.

Why are arrays important?
They organize collections, allow iteration and efficient access to sequences of data.

Why are objects important?
They model real-world entities with properties and behaviors and group related data logically.

What is an arrow function?
A concise ES6 function syntax => that has lexical this binding and shorter form.

What is JSON?
JavaScript Object Notation, a text format for structured data interchange using key-value pairs and arrays
