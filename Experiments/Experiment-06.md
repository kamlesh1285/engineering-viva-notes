Experiment 6: DOM Manipulation and Event Handling using JavaScript

What is DOM?
The Document Object Model, a tree-like representation of an HTML/XML document that scripts can interact with.

What does DOM stand for?
Document Object Model.

What is DOM Manipulation?
Changing the document structure, style, or content using JavaScript (e.g., adding elements, updating text).

What is an event in JavaScript?
An action or occurrence (click, input, load) that code can respond to.

What is Event Handling?
The process of listening for events and executing callback functions when they occur.

What is addEventListener()?
A method to attach an event listener to an element for a specified event type.

What is getElementById()?
A DOM method that returns the element with the specified id attribute.

What is the use of textContent?
To get or set the textual content of an element without parsing HTML.

What is the use of innerHTML?
To get or set an element’s HTML content, which will be parsed and rendered.

How can CSS be changed using JavaScript?
By modifying element.style properties or by adding/removing CSS classes.

What is the click event?
An event fired when a pointing device button is pressed and released on an element.

What is the mouseover event?
An event that fires when the pointer is moved onto an element or one of its children.

What is the change event?
An event that fires when the value of an input, select, or textarea is changed and loses focus (or immediately for some controls).

Why is DOM important?
It provides a programmable interface to interact with and update web pages dynamically.

What is user interaction?
Any input or action from the user (clicks, typing, gestures) that affects application behavior.s
