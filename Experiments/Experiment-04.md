Experiment 4: JavaScript Program using Variables, Conditions, and Loops

What is JavaScript?
A high-level programming language primarily used to create interactive behavior in web pages and applications.

What is a variable?
A named container for storing values that can be read or changed during program execution.

What are JavaScript data types?
Primitive types: string, number, boolean, null, undefined, symbol, bigint; and objects (including arrays, functions).

What is a loop?
A construct that repeats a block of code multiple times until a condition is met.

What is a for loop?
A loop with initialization, condition, and increment expressions used for counted iteration.

What is a conditional statement?
A statement (if/else/switch) that executes code branches based on boolean conditions.

What is an if-else statement?
It runs one block when a condition is true and another block when it is false.

What is the purpose of prompt()?
To display a dialog that asks the user for input and returns the entered string (in browsers).

What is parseInt()?
Converts a string to an integer, optionally using a specified radix.

What is parseFloat()?
Converts a string to a floating-point number.

How is average calculated?
Sum of values divided by the number of values: average = total / count.

What is the purpose of loops?
To automate repetitive tasks and iterate over data collections.

What is the difference between let, var, and const?
var is function-scoped and hoisted; let is block-scoped and mutable; const is block-scoped and read-only reference (cannot be reassigned).

What is dynamic output?
Content generated or updated at runtime (e.g., DOM changes) in response to data or user actions.

Why are conditions used in programming?
To control flow and make decisions so different code runs for different situations
