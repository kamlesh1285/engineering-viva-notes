Experiment 1: Responsive Web Page using Bootstrap Grid System

What is Bootstrap?
A popular open-source CSS framework that provides prebuilt components, layout utilities, and JavaScript plugins to speed up responsive web development.

What is responsive web design?
A design approach that makes web pages adapt layout and content to different screen sizes and devices for good usability and readability.

What is the Bootstrap Grid System?
A responsive layout system using rows and columns based on a 12-column flexible grid to arrange page content.

How many columns are available in Bootstrap Grid?
12 columns.

What is a container in Bootstrap?
A wrapper that centers page content and provides horizontal padding; available as .container (fixed-width at breakpoints) and .container-fluid (full width).

What is the purpose of a row?
To group columns and clear horizontal gutters; rows ensure columns align properly using negative margins.

What is the purpose of a column?
To hold content and define its width across breakpoints using classes like .col-*, forming the grid layout.

What are Bootstrap breakpoints?
Named viewport widths where layout behavior changes (e.g., xs, sm, md, lg, xl, xxl) used to apply responsive rules.

What is the difference between col-sm, col-md, and col-lg?
They target different minimum viewport widths: col-sm applies from small screens and up, col-md from medium and up, col-lg from large and up, controlling when column sizing takes effect.

Why is Bootstrap called a mobile-first framework?
Its CSS is written for small screens first, then uses min-width breakpoints to progressively enhance layouts for larger screens.

What is Bootstrap CDN?
A content delivery network hosting Bootstrap CSS/JS files so you can include them via remote URLs instead of local files.

What are the advantages of using Bootstrap?
Faster development, responsive grid, consistent components, cross-browser compatibility, good documentation, and a large ecosystem.

What is the purpose of the viewport meta tag?
To control layout on mobile browsers by setting the viewport width and scale so responsive CSS works correctly.

How can you make a webpage responsive?
Use fluid layouts, relative units, media queries or framework utilities, responsive images, and test across breakpoints.

What is the use of the container-fluid class?
It creates a full-width container that spans the entire viewport width, useful for edge-to-edge layouts
