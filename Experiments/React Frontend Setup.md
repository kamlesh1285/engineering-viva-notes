React Frontend Setup
Create a React Project (Vite - Recommended):-

npm create vite@latest smart-attendance-frontend -- --template react

Move into project:-

cd smart-attendance-frontend

Install dependencies:-

npm install

Start development server:-

npm run dev

Build for production:-

npm run build

Preview build:-

npm run preview
Common React Commands

Install React Router:-

npm install react-router-dom

Install Axios:-

npm install axios

Install Bootstrap:-

npm install bootstrap

Install Tailwind CSS:-

npm install tailwindcss @tailwindcss/vite
