Experiment 3: Mobile-First Responsive Design using Tailwind CSS

What is Tailwind CSS?
A utility-first CSS framework that provides low-level utility classes to build custom designs without pre-styled components.

What is utility-first CSS?
An approach that offers single-purpose utility classes (like .text-center, .p-4) to compose styles directly in HTML.

What is mobile-first design?
Designing for the smallest screens first and progressively enhancing for larger screens, ensuring core functionality on mobile.

What are responsive prefixes in Tailwind?
Short class prefixes (sm:, md:, lg:, etc.) that apply utilities at and above specified breakpoints.

What does sm: represent?
Applies the following utility at the small breakpoint and up (default Tailwind small breakpoint ~640px).

What does md: represent?
Applies the utility at the medium breakpoint and up (default ~768px).

What does lg: represent?
Applies the utility at the large breakpoint and up (default ~1024px).

What is the use of flex in Tailwind?
The .flex utility makes an element a flex container so you can control layout with flexbox utilities.

What is the use of grid in Tailwind?
The .grid utility makes an element a CSS Grid container to define rows/columns and place items precisely.

What is container mx-auto?
.container sets a responsive fixed-width wrapper; .mx-auto centers it horizontally by applying automatic side margins.

What does font-bold do?
Applies a bold font weight to text.

What does text-xl mean?
Sets the font size to Tailwind’s predefined extra-large size for readable, larger text.

What is the purpose of spacing utilities?
To control margin, padding, gap, and spacing consistently using small, composable classes.

Difference between Bootstrap and Tailwind CSS?
Bootstrap provides pre-styled components and a design system; Tailwind provides low-level utility classes for building custom designs with minimal opinionated styling.

Why is responsive design important?
It ensures websites are usable and accessible across devices, improving user experience and reach.
