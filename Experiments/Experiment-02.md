Experiment 2: Using Bootstrap UI Components for Web Page Design

What is a Navbar in Bootstrap?
A responsive navigation component that contains links, brand, toggler, and optional forms or dropdowns.

What is a Card component?
A flexible content container with options for header, body, footer, images, and utilities for consistent styling.

What are Bootstrap Buttons?
Pre-styled button classes (e.g., .btn, .btn-primary) that provide consistent sizing, colors, and states.

What is a Badge in Bootstrap?
A small count or label component used to display numbers or status over items.

What is a Modal?
A dialog component that overlays the page to display content or prompt user interaction without leaving the current page.

What is the purpose of a Modal popup?
To capture user attention for tasks like confirmations, forms, or important messages while preventing interaction with the background.

What is the use of navbar-expand-lg?
It makes the navbar collapsed (hamburger) on viewports smaller than the large breakpoint and expanded on large screens and up.

What is the difference between a Card and a Container?
A Card is a styled content box component for grouping content; a Container is a layout wrapper that controls page width and padding.

What is the use of btn-primary?
It applies the primary color styling to a button to indicate primary actions or emphasis.

How do you open a modal using Bootstrap?
Use data attributes (data-bs-toggle="modal" data-bs-target="#id") on a trigger element or call the modal via JavaScript API (new bootstrap.Modal(...).show()).

What is the purpose of Bootstrap Icons?
A free icon set designed to match Bootstrap’s style, used to add scalable vector icons to interfaces.

Why are UI components important?
They provide consistent, reusable building blocks that speed development and improve usability and accessibility.

What is the role of CSS in Bootstrap?
CSS provides the styling, layout, and responsive behaviors for Bootstrap’s components and utilities.

How does Bootstrap improve development speed?
By offering prebuilt components, utility classes, and a consistent design system that reduces the need to write CSS from scratch.

What are responsive components?
Components designed to adapt their layout and behavior across different screen sizes (e.g., responsive navbars, cards that stack).
